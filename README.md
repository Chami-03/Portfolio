This repository hosts a clean, modern, single-page portfolio website for Chamikara Wijerathne, an IT undergraduate specializing in Business Analysis and Data Analysis. The design features a blend of Neumorphism and Glassmorphism, complete with a dynamic, theme-toggling interface and an interactive 3D skill cube.

üåü Key Features
Responsive Design: Optimized for desktop, tablet, and mobile screens.

Theme Toggle: Supports a smooth toggle between Light Mode (Neumorphism) and Dark Mode (Glassmorphism).

Interactive 3D Skill Cube: A Three.js powered element in the hero section.

Hover or Drag to rotate.

Click or Tap the cube to display a short note about the featured skill (BA, DA, SQL, etc.).

Single-Page Layout: Smooth scrolling navigation to sections: About, Projects, and Contact.

Simple AI Assistant: A local JavaScript function provides structured advice (BA Case Study, SQL Checklist) for common BA/DA prompts.

Scroll Reveal Effects: Content fades and slides in as the user scrolls down.

Resume Download: A button is included to download a text file version of the resume.
